Folders:
- Photos: photos (dorsal and lateral) of adult mayflies. Filenames correspond to Ind.code in the dataset.
- Data analysis, with
	* In main folder: scripts for 
		* Maps (Map.Rmd), 
		* Stats of the Hierdense beek based on environmental data recorded in the field and weather data (Stats_Hierdensebeek.Rmd)
		* Trait analysis (cleaning the data and then analyzing, Traits - 01 and 02. Rmd).
	* Data: contains
		* measurements of E. danica individuals, including a readme file.
		* environmental data recorded in the field, including a readme file.
	* GIS: 
		* Layers of the Netherlands, Dutch streams, Hierdensebeek and sampling locations.
		* Land use categorization
	* Weather:
		* Weather data from KNMI and in-stream loggers
